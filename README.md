
# How to run the project
First you need to create a virtual environment
```bash
python3 -m venv venv 
```
Activate the venv
```bash
source venv/bin/activate
```
Install the dependencies
```bash
pip install -r requirements.txt
```
And you can run the code by running command below:
```bash
python main.py
```
